# Computer Types

**[View Live Site](https://ritapereira10.github.io/computer-types/)**

A guide to different types of computers and their characteristics.

## Overview

This project explores various categories of computers, from personal devices to enterprise systems.

## Categories

- **Personal Computers** - Desktops and laptops for individual use
- **Servers** - Systems designed to provide services to other computers
- **Workstations** - High-performance machines for specialized tasks
- **Embedded Systems** - Computers built into other devices
- **Supercomputers** - The most powerful computing systems

## Development

This project uses [Vite](https://vitejs.dev/) for fast development and building.

### Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
